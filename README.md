# WoT Shop — Галерея аккаунтов Мир Танков

Статический сайт-галерея для продажи игровых аккаунтов на FunPay.  
Работает полностью на GitHub Pages, без сервера.

---

## Развёртывание на GitHub Pages

### 1. Создайте репозиторий

Создайте новый публичный репозиторий на GitHub, например `wot-shop`.

### 2. Загрузите файлы

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/wot-shop.git
git push -u origin main
```

### 3. Включите GitHub Pages

- Перейдите в **Settings → Pages**
- Source: **Deploy from a branch**
- Branch: **main**, папка **/ (root)**
- Нажмите **Save**

Сайт будет доступен по адресу: `https://YOUR_USERNAME.github.io/wot-shop/`

## Настройка GitHub токена

### 1. Создайте Personal Access Token

- Перейдите: https://github.com/settings/tokens
- Нажмите **Generate new token (classic)**
- Название: `wot-shop-admin`
- Выберите права: ✅ `repo` (полный доступ к репозиторию)
- Нажмите **Generate token**
- **Скопируйте токен** — он показывается только один раз!

> Для дополнительной безопасности: вместо `repo` можно выбрать **Fine-grained token**  
> с доступом только к вашему репозиторию и правами `Contents: Read and write`.

### 2. Введите токен в админке

- Откройте `/admin/` на вашем сайте
- Нажмите **⚙ Настройки**
- Введите токен
- Введите репозиторий в формате `username/repo-name`
- Нажмите **Сохранить и проверить**

Токен сохраняется в `localStorage` браузера — вводить его нужно только один раз на каждом устройстве.

## Работа с админкой

### Управление витринами

1. **Создать витрину** — нажмите `+ Добавить витрину` в сайдбаре
   - Введите уникальный ID (латиница, например `shop1`)
   - Введите название и описание
   
2. **Редактировать** — выберите витрину → кнопка `✏ Изменить`

3. **Удалить** — кнопка `🗑 Удалить витрину` (удаляет данные и JSON)

### Управление лотами

1. **Создать лот** — кнопка `+ Добавить лот`
   - Введите название
   - Введите ссылку на объявление FunPay

2. **Редактировать** — кнопка `✏` рядом с лотом

3. **Удалить** — кнопка `🗑` (удаляет лот и все его изображения)

### Загрузка изображений

1. Нажмите `🖼` рядом с лотом — откроется менеджер изображений

2. **Загрузить** — перетащите файлы в зону или нажмите `+ Добавить фото`
   - Поддерживаются PNG, JPG, GIF, WebP
   - Изображения автоматически конвертируются в WebP
   - Загружаются прямо в репозиторий через GitHub API

3. **Сортировка** — перетаскивайте карточки или используйте кнопки `↑↓`

4. **Удаление** — кнопка `🗑` на каждой карточке

## Просмотр галереи

Пользователи открывают лот и видят сетку скриншотов.  
Нажатие на любое изображение открывает лайтбокс:

- **Стрелки** или клавиши ←→ — переключение
- **Скролл мыши** или `+/-` — масштабирование
- **Двойной клик** — быстрое приближение/отдаление
- **Перетаскивание** — панорамирование при зуме
- **Свайп** — переключение на мобильных
- **ESC** — закрыть

## Несколько продавцов

Для каждого аккаунта FunPay создайте отдельную витрину:

- `shop1` → ваш первый аккаунт FunPay
- `shop2` → второй аккаунт FunPay

Покупатели с FunPay переходят по прямой ссылке на витрину  
и видят только её лоты.
